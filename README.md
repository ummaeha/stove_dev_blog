# 실행하는 방법
1. 본 레포지토리를 clone 받는다.
2. 터미널을 켜고 아래의 명령어를 차례로 실행한다.
```
cd backend
npm install
npm start

cd database
npm install

cd client-web 
npm install
npm start

```
3. http://localhost:3000/ 에 접속한다. 🥳

# 포트 정보
3000: Client Server(Reactjs)
4000: Nodejs Express Backend Server
4001: JSON-server, db.json
